# virtualbook

# App purpose:

Create and manage the scheduling of Black Tie Rental of Florida company.__

Two main modules:__

*Settings (create, update, search and delete):_
  -Users_
  -Clients_
  -Locations_
  -Items__
  
*Events:_
  -Creation Form_ 
  -Calendar View by month__
  
Create by Cristian Ruiz Gonzalez @Cruizg93__
